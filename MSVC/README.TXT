This document decribes the release notes for version 1.5 of Microsoft
Visual C++.

The majority of the release notes for Visual C++ are in Windows Write format. 
You can view these files using the Technote Viewer. There are four files 
containing release information.

README.WRI
Contains release notes and important information about Visual C++.

ERRATA.WRI
Lists errors in the documentation. Some information is duplicated in the
README.WRI and ERRATA.WRI files.

CONFIG.WRI
Contains information that may be needed to configure Visual C++ to work
properly on some systems.

TIPS.WRI
Contains additional information to help you use Visual C++ more effectively.

PROBRPT.WRI
Template for reporting problems with Visual C++. To report a problem, print this
form, fill in the relevant information, and mail or fax it to Microsoft.

README.HLP
Used by the Technote Viewer.

     NOTE: Microsoft improves its Visual C++ documentation at the time of
     reprinting, so some of the information in this file may already be
     included in your manuals.
